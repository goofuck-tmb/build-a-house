# 📊 Счётчики

Снимай показания **1-го числа каждого месяца**.  
Это даст чистую статистику потребления.

## Файлы

| Файл | Ресурс |
|------|--------|
| [electricity.csv](electricity.csv) | Электричество (Т1/Т2) |
| [water-cold.csv](water-cold.csv) | Холодная вода |
| [water-hot.csv](water-hot.csv) | Горячая вода |
| [gas.csv](gas.csv) | Газ |

## Пример заполнения

```csv
дата,показание,расход,сумма_руб,примечание
2025-01-01,1000,,, первое показание
2025-02-01,1085,85,680,
2025-03-01,1210,125,1000, морозы
```

## Анализ потребления

После года данных можно:
- Построить графики расхода по месяцам
- Сравнивать год к году
- Замечать аномалии (утечка? неисправность котла?)
