## Сведения из полной выписки ЕГРН

### Основное
- Площадь: **794 м² ± 10** (7,94 сотки)
- Размеры: **~38,5 × ~22 м** (трапеция, стороны: 38.5 / 21.83 / 37.98 / 21.95 м)
- Категория: Земли населённых пунктов, ИЖС ✅
- Обременения (залог, арест): **не зарегистрированы** ✅
- Правообладатель: физическое лицо, собственность с 20.09.2021 ✅

| Зона | Подзона | Ограничение | Влияние на стройку |

| 68:00-6.651 | 5-я подзона | Запрет опасных производственных объектов | ❌ Не касается ИЖС |
| 68:00-6.798 | 5-я подзона | Запрет опасных производственных объектов | ❌ Не касается ИЖС |
