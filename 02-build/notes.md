## 🏗️ Технические решения

### Отмостка — мягкая
Сделать мягкую (скрытую) отмостку вместо бетонной:
- Профилированная мембрана (типа DELTA-DRAIN) + геотекстиль + щебень + финишный слой (газон, плитка)
- Для свайного фундамента особенно актуально — бетонная отмостка при морозном пучении может деформироваться
- Лучше отводит воду, не трескается, дешевле в ремонте
